Destination

Selectionnez le fichier dans lequel la liste sera generée.

Enfin selectionnez le type de liste a générer.

Trois types sont disponibles:
tsv:
Un simple fichier texte dans lequel les différents informations sont séparées par des tabs (tabulations)
xml:
Un fichier xml
autre:
Un autre type a la prochaine étape (html,csv,..).
