Seleccione el destino

Ahora debe seleccionar el fichero donde se escribirá el listado.

Por último, seleccione el formato del listado que desea crear.

Actualmente existen tres opciones:
tsv:
Crea un fichero de texto con una línea por cada fichero de música. El contenido de las etiquetas de cada fichero de música se separa por tabulaciones
xml:
Crea un fichero xml
other:
Permite seleccionar otro formato en la siguiente pantalla.
